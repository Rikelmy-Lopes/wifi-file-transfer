<h1 align="center" id="title"> Wifi File Transfer</h1>

<p id="description"> Wifi File Transfer is a free and open-source program built with Rust, TypeScript, Tauri, and React that allows you to transfer files between devices connected to the same Wi-Fi network. It is an easy and convenient way to share files between computers, laptops, smartphones, and tablets.

To use Wifi File Transfer, simply install the program on the device that will transfer the files. Next, open a web browser on the destination device and type the source device's IP address into the address bar. You can then select the files you want to transfer. The files will be transferred automatically to the destination device. </p>

<h2> Prerequisites </h2>

<p>Before you begin, you will need to have the following tools installed on your machine: </p>

- **[Nodejs](https://nodejs.org/en/)** >= 18
- **[Rust](https://rust-lang.org/)** >= 1.90
- **[Git](https://git-scm.com)**

<h2>üõ†Ô∏è Install:</h2>

<p>1. Clone this repository to your local directory using the following command:</p>

```
git clone git@github.com:Rikelmy-Lopes/wifi-file-transfer.git
```

<p>2. Navigate to the project directory:</p>

```
cd wifi-file-transfer
```

<p>3. Install Dependencies:</p>

```
npm install
```

<p>4. To enter Development Mode:</p>

```
npm run dev
```

<p>5. To create an Application Build</p>

```
npm run build
```
